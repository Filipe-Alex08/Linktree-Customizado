# Linktree Personalizado

Linktree profissional desenvolvido com HTML, CSS e JavaScript para centralizar informações de carreira, projetos, redes sociais e formas de contato em uma única página moderna e responsiva.

## Funcionalidades

- Perfil profissional
- Links para LinkedIn, GitHub e WhatsApp
- Integração com portfólio
- Seção de competências
- Apresentação de projetos
- Layout responsivo
- Interface moderna e intuitiva

## Tecnologias

- HTML5
- CSS3
- JavaScript

## Objetivo

Criar uma página profissional personalizada que funcione como um cartão de visitas digital, facilitando o acesso às minhas informações, experiências e projetos.
